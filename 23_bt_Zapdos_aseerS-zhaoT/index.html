Containers

This text is in a container. Elements, such as rows, go into columns for ease of display and control.
Another important note is that containers have padding on either side.


Rows and Columns

In Bootstrap, rows go into containers. Each row can be can be dividedinto sections ranging from 1 to 12. These sections are called "columns".
This can be done with the following code:

          <div class="container">
              <div class="row">
                  <div class="col-sm">
                      The number of times this piece is repeated is the number of columns
                  </div>
              </div>
          </div>

Buttons

Bootstrap has many decorative styles of buttons that can be easily created. Of course, all of these buttons are highly customizable. Buttons are done with the following code by replacing the * with their names:

<button type="button" class="btn btn-*">ButtonName</button>

These are some of the different colored styles by default. They are labeled by their names.


Some other button styles included by default are:

	    Outline: <button type="button" class="btn btn-outline-primary">Primary</button>
	  
	    Disabled: <button type="button" class="btn btn-outline-primary">Primary</button>

Media

Bootstrap allows for smooth implementation of media next to a content block. Media can be images, video, audio.

Media Heading

This is the media body (the text formatted next to the media.
The code for media is:

	      <div class="media">
	          Media content here
	          <div class="media-body">
	              <h* class="media-heading">Media heading here</h*>
	          </div>
	      </div>


Alerts

Bootstrap provides many default alerts. Alerts are text enclosed within a box, making them super easy to manipulate in terms of front-end displaying. They can be created using the following code:

	  <div class="alert alert-*" role="alert">primary alert</div>

where * is replaced by the alert style's name. Some alerts are shown below with their names.

Tables

Bootstrap provides easy access to make a table. The following code is used to make a table. You can have as many th, tbody, and td as you need.

	  <table class="table">
	      <thead style=Add Styling Here>
	          <th scope="col">ColumnName</th>
	          ...
              </thead>
	      <tbody>
	          <td>Name</td>
	          ...
	      </tbody>
	      <tbody>
	      ....
	      <tbody>
	  </table>

Jumbotron

Jumbotron is Bootstrap providing a way to really bring attention to a particular section of the screen. It emphasizes large and can be created with the following code:

	  <div class="jumbotron">
	    Content here
	  </div>
